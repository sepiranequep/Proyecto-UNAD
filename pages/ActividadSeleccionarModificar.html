  1. ¿Qué sentencia del lenguaje SQL se utiliza para modificar un registro?
    1. SELECT
    2. DELETE
    3. UPDATE
    4. LIKE

  2. ¿Qué datos se requieren conocer para realizar correctamente la sentencia SQL que permita la modificación de un registro?
    1. HostName y Password.
    2. Nombre de la tabla y Id del registro.
    3. Todos los anteriores.
    4. Ninguno de los anteriores.

  3. La palabra reservada WHERE del lenguaje SQL, se utiliza para:
    1. Eliminar un registro de una tabla.
    2. Ordenar los registros de mayor a menor.
    3. Unir dos tablas sin que aparezcan datos repetidos.
    4. Distinguir uno o varios registros de una tabla, para que solo estos sean afectados por alguna operación a realizar.

    3. "UPDATE nombreTabla SET nombreAtributo1='...',nombreAtributoN='...'".
    4. Ninguna de las anteriores.

  5. ¿Con que palabra reservada del lenguaje de programación PHP podemos realizar la ejecución de una sentencia SQL?
